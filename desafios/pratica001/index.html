A história do HTML5

O HTML5 é a quinta versão do HTML, que é a linguagem de marcação para sites na web.
Muitas vezes a linguagem HTML é confundida com uma linguagem de programação, o que está completamente errado, pois o HTML é apenas uma linguagem de marcação, se diferenciando completamente das linguagens de programação, como Python por exemplo.

O HTML5 e o CSS3 são como "irmãos" junto do JavaScript, um site que tenha apenas o HTML5 seja usado terá o texto, imagens e etc, mas não terá a beleza das interações propostas pelo JavaScript e muito menos as edições de fonte do próprio texto do site que o CSS3 permite ao desenvolvedor do site fazer.
